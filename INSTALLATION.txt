================================================================================
  ALGIERS UNIVERSITY ATTENDANCE MANAGEMENT SYSTEM
  Installation Instructions
================================================================================

SYSTEM REQUIREMENTS:
- PHP 7.4 or higher
- MariaDB 5.7 / MySQL 5.7 or higher
- Apache or Nginx web server
- Modern web browser
- 10 MB free disk space

================================================================================
STEP 1: EXTRACT PROJECT
================================================================================

Extract the project files to your web server directory:
- Windows WAMP: C:\wamp64\www\PawProject\
- Windows XAMPP: C:\xampp\htdocs\PawProject\
- Linux LAMP: /var/www/html/PawProject/
- macOS MAMP: /Applications/MAMP/htdocs/PawProject/

================================================================================
STEP 2: VERIFY WEB SERVER
================================================================================

Ensure Apache/Nginx is running:
- WAMP: Start WAMP, ensure all services are green
- XAMPP: Start Apache and MySQL from Control Panel
- Terminal: Start your server normally

================================================================================
STEP 3: INITIALIZE DATABASE
================================================================================

Method 1 - Web Browser (RECOMMENDED):
1. Open your browser
2. Navigate to: http://localhost/PawProject/setup.php
3. You should see: "Database Setup Complete!"
4. The default admin account will be created

Method 2 - Command Line:
1. Open terminal/command prompt
2. Navigate to: cd C:\wamp64\www\PawProject\
3. Run: php setup.php
4. Look for: "Database Setup Complete!"

================================================================================
STEP 4: VERIFY DATABASE
================================================================================

Check that database was created:
1. Open phpMyAdmin: http://localhost/phpmyadmin/
2. Login (usually no password for root)
3. Look for database named: "algiers_attendance"
4. Verify tables exist: users, courses, sessions, attendance, etc.

If not created, check:
- MySQL/MariaDB is running
- Port 3306 is accessible
- Default credentials are used

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Connection failed" error
SOLUTION:
- Verify MySQL/MariaDB is running
- Check database credentials in includes/db.php
- Run setup.php again to recreate database

PROBLEM: Login page loops back to login
SOLUTION:
- Clear browser cache and cookies
- Try private/incognito browser window
- Check PHP session directory permissions

PROBLEM: Can't upload files
SOLUTION:
- Create folder: uploads/justifications/
- Set permissions: chmod 755 uploads/
- Ensure file size < 5MB

PROBLEM: Charts not showing on statistics page
SOLUTION:
- Check browser console for JavaScript errors
- Verify Chart.js CDN is accessible
- Check browser allows third-party scripts

PROBLEM: AJAX attendance saving fails
SOLUTION:
- Check browser console for errors
- Verify database connection working
- Check ajax_save_attendance.php exists
